Installation
============

Requirements
------------

- Python >= 3.8
- PyTorch >= 2.1

Install from source
-------------------

.. code-block:: bash

    git clone https://github.com/intsystems/implicit-reparameterization-trick.git
    cd implicit-reparameterization-trick
    pip install src/

Uninstall
---------

.. code-block:: bash

    pip uninstall irt
